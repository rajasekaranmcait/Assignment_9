# Assignment_9:


To Write a simple android program to display the message on an android activity about the data connectivity of the device. 
When the data connectivity is switched off the message should say “internet connectivity not available” 
and when the data connectivity is available the message should say “internet connectivity is available”
To use the android studio IDE for this project.

Context: As the name suggests, its the context of current state of the application/object. 
It lets newly created objects understand what has been going on. 
Typically you call it to get information regarding another part of your program (activity, package/application)

Main Activity : Its the frontview or main screen of the android mobile screen.
Manifest : Its used to connect permission for mobile data or wife internet connection
BroadcastReceiver : prebuild the application operations for android already.
